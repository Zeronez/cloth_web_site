AnimeAttire Production Orchestration Plan
=========================================

Status Statement
----------------
Current state is NOT a complete production product yet.
It is a strong engineering foundation: repository structure, Django/DRF core, SKU model,
cart/checkout service, Next.js storefront scaffold, Docker, and CI skeleton exist.

Definition of Done for "production-ready"
-----------------------------------------
[ ] A real customer can register, browse, filter, add SKU variants to cart, pay, receive order confirmation, track fulfillment, and get support.
[ ] Store staff can manage catalog, stock, orders, refunds, shipping statuses, content, discounts, and customer issues without developer access.
[ ] Payments, delivery, email/SMS, media storage, analytics, monitoring, backups, secrets, CI/CD, security controls, and legal pages are wired for real operation.
[ ] Every release passes automated tests in CI against real service containers: PostgreSQL, Redis, Celery worker, object storage emulator, frontend browser, and API.
[ ] The site is Russian-first for CIS audience: Russian copy, prices in RUB by default, regional checkout UX, Russian legal/support pages, and localized emails.
