# AnimeAttire Frontend

Production-ready Next.js App Router storefront scaffold for AnimeAttire.

UI stack:

- TypeScript strict mode.
- Tailwind CSS.
- Zustand for persisted cart client state.
- React Query for API caching.
- Framer Motion for hero reveal and cart drawer motion.

The app source tree lives under `app/`, `components/`, `lib/`, and `stores/`.
